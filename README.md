
# Project Title

Project for MYSQL+ERN stack task list with CRUD operation


## Tech Stack

**Client:** react, bootstrap 4, postman

**Server:** node, express, nodemon, cors, mysql2, mysql DB

  
## Features

- Add new task
- View all task
- Update task'/modal
- Delete task
